# 重訓紀錄網站

Demo 網址: 
https://workout-records-demo.fly.dev/


功能敘述
* 後端是用php laravel，前端是用vue
* 記錄重訓紀錄
* 目前註冊不驗證email，也沒有重設帳密功能
